# ExpirySaver
ExpirySaver is a platform designed to help reduce food waste by connecting users with discounted surplus food. 

## Features
- Discover discounted food offers from local stores.
- Responsive and mobile-friendly design.

## Installation
1. Clone the repository:
2. Open `index.html` in your browser.
